# Agent Org Network

개인별 업무 에이전트를 만들고, 중앙 라우터가 “누가 무엇을 담당하는지”를 기준으로 질문과 일을 연결하는 조직형 멀티에이전트 네트워크 실험 프로젝트.

## 한 줄 설명

중앙 AI가 모든 걸 아는 구조가 아니라, 각 개인/팀의 업무 에이전트가 자기 전문 영역을 책임지고 중앙은 담당자, 권한, 라우팅, 감사 기록만 관리한다.

## 작업 가설

기존 멀티에이전트 프레임워크는 많지만, “조직 구성원마다 자기 업무 에이전트를 운영하고 중앙이 담당자/권한/신뢰도 기반으로 연결한다”는 제품 철학은 아직 명확한 대표 오픈소스가 없다. A2A Agent Card, capability registry, handoff 프레임워크를 엮으면 작게 검증할 수 있다.
