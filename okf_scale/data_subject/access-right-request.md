# 개인정보 열람 요구권

개인정보보호법 제35조 제1항에 따라 정보주체는 개인정보처리자가 처리하는 자신의 개인정보에 대한 열람을 해당 개인정보처리자에게 요구할 수 있다. 열람 요구를 받은 개인정보처리자는 대통령령으로 정하는 기간 내에 열람할 수 있도록 해야 한다(제35조 제3항).
