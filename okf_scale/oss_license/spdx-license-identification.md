# SPDX 식별자를 통한 라이선스 확인

SPDX license-list-data 저장소가 배포하는 MIT, Apache-2.0, GPL-3.0-only 텍스트는 각각 고유한 SPDX 식별자로 특정된다. 소스 파일 헤더에 `SPDX-License-Identifier: MIT`처럼 표기하면 해당 파일이 어떤 라이선스 원문 조건을 따르는지 기계 판독 가능하게 밝히는 것이며, 라이선스 원문 자체는 축어적 복제·배포가 허용된다.
