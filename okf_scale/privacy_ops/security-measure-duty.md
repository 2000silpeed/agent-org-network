# 개인정보 안전성 확보 조치

개인정보보호법 제15조 제3항·제17조 제4항은 정보주체의 동의 없이 개인정보를 이용·제공할 수 있는지를 판단할 때 암호화 등 안전성 확보에 필요한 조치를 하였는지 여부를 고려 요소로 명시한다. 안전조치 수준이 낮으면 동의 없는 이용·제공의 특례 요건을 충족하기 어려워지므로, 안전조치는 개인정보 처리 전반에서 선행되어야 할 기본 조치로 다뤄진다.
